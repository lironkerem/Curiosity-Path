# Curiosity Path

**A Progressive Web Application for Conscious Living and Self-Realization**

Curiosity Path serves as a comprehensive digital companion for personal growth, mindfulness, emotional integration, and spiritual exploration. The platform integrates timeless wisdom practices with contemporary tools — including gamification, artificial intelligence, immersive environments, and advanced self-analysis — to support meaningful inner development.

[Visit Live Application →](https://digital-curiosity-path.vercel.app)

## Core Capabilities

### Mindfulness & Emotional Wellness
- Energy Tracking and Reflection
- Gratitude and Affirmations Systems
- Journaling Environment
- Meditation Support
- Happiness Cultivation and Self-Inquiry Tools
- Daily Inspirational Guidance

### Transformative Spiritual Practices
- Comprehensive Tarot System with AI-enhanced interpretation (TarotVisionAI)
- Shadow Work Framework (ShadowAlchemyLab) — including archetype exploration and guided processes
- Multi-Modal Self Analysis (SelfAnalysisPro) — integrating astrology, numerology, tarot, and professional PDF reporting

### Immersive Experiences
- **Community Hub**: Virtual gathering space featuring seasonal and thematic rooms:
  - Lunar Cycle Rooms (Full Moon, New Moon, Waxing, Waning)
  - Solar Seasonal Rooms (Spring, Summer, Autumn, Winter)
  - Specialized Practice Rooms (Breathwork, Reiki, OSHO, Campfire, Silent Meditation, Chakra Work, and more)
- Narrative Reframing Tool (FlipTheScript)
- Karma Economy and Reward System (Karma Shop)

### Platform Foundations
- Sophisticated gamification and progress architecture
- Conversational AI guidance
- Dark mode and fully responsive interface
- Progressive Web App (PWA) capabilities for seamless cross-device access

## Technology Stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **Build & Tooling**: Vite (in active migration for improved architecture)
- **Authentication & Data**: Supabase with Google OAuth
- **Deployment**: Vercel
- **Additional**: Service Worker and Web App Manifest for PWA functionality
